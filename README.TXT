------------------------------------------------------------------------------------------------------------
	FACE RECOGNITION PACKAGE
------------------------------------------------------------------------------------------------------------

The contents under the folder named FaceRecognition Final must be in the path 
C:\j2sdk1.4.2_05\bin. 

To run the program, at the command prompt
 C:\j2sdk1.4.2_05\bin>java Face <url> <frame_rate> <output_directory> 
must be entered.

<url> 		- the location of the input mpeg file
<frame_rate> 	- must be less than the framerate of the mpeg file(usually 30
		     frames/sec.)
		- numerical value, prefererably integers.
<output_directory>	- the location where the output jpg files must be written into.

The folder also contains some files which have been used for testing and the
 results obtained through those testing.
